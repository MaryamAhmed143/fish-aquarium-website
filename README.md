# 🐠 Fish Aquarium Website

Welcome to my Fish Aquarium Website project!

## 📖 About

This project is a responsive website created using HTML, CSS, and JavaScript. It provides information about different fish species, aquariums, and aquarium features with a modern and attractive design.

## ✨ Features

- 🐟 Beautiful homepage
- ℹ️ About section
- 🐠 Aquarium collection
- ⭐ Features section
- ⚙️ Process section
- 💬 Reviews section
- 📱 Responsive design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
